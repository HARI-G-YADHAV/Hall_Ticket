## Tech Stack

## Run Locally

Clone the project

```bash
  git clone https://github.com/HARI-G-YADHAV/Hall_Ticket.git
```

Go to the project directory

```bash
  cd 
```

Install Required Packages 

```bash
  pip3 install -r requirement.txt
```

Start the server in debug mode

```bash
  pip3 app.py

```
